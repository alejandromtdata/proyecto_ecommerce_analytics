# Dashboard E-Commerce - Power BI

## Descripción del proyecto

Dashboard interactivo desarrollado en Power BI para analizar el rendimiento comercial de un e-commerce, incluyendo ventas, clientes, productos y evolución temporal.

## Objetivos

- Analizar ventas totales
- Identificar clientes principales
- Comparar ventas por país y categoría
- Detectar tendencias mensuales
- Diseñar un dashboard ejecutivo e interactivo

## KPIs principales

- Ventas Totales
- Número de Ventas
- Clientes Únicos
- Ticket Medio

## Herramientas utilizadas

- Power BI
- Power Query
- DAX
- CSV / Excel

## Principales conclusiones

- Francia lidera ligeramente el volumen total de ventas.
- Las ventas entre países muestran una distribución relativamente equilibrada.
- Febrero presenta el menor volumen de ventas mensual.
- La categoría “Dispositivos” representa la mayor parte de las ventas.
- Existe un grupo reducido de clientes que concentra una parte importante de los ingresos.
