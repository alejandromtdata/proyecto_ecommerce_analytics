# PROYECTO ECOMMERCE ANALYTICS 

Proyecto de analisis de datos de un E-commerce utilizando las herramientas: 

- Github 
- Python
- SQL (PostgreSQL)

## Objetivo del proyecto: 

- Aanalizar el rendimiento del e-commerce a traves de KPI's: 

- Ventas totales
- Ventas por pais y por su categoria
- Crecimiento
- Clientes únicos
